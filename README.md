# Newsweek Clone

Microverse project to learn responsive design and bootstrap

![screenshot](./images/screenshot.jpg)

The page tries to replicate as much as possible the actual home page of newsweek.com.

## Built With

- Html,
- CSS,
- Bootstrap
- Font Awsome for icons
